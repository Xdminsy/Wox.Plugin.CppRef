## 安装步骤

### 安装Wox

[github链接](https://github.com/Wox-launcher/Wox)



### 安装CppRef


#### 方法一（建议）：

到Release中下载插件包（可以获取最新版本），解压到wox主题文件夹中并重启wox

#### 方法二：

打开wox，输入命令

```
wpm install CppRef
```
